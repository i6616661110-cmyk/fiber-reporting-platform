# Fiber Field-Reporting Platform

A web platform built for a fiber-optic splicing team to log completed work directly from the field, centralize data in one place, and export it in the client's required format.

## Impact

- Cut weekly report preparation from 3–5 hours to under 1 hour
- Gave management real-time visibility into field progress
- Handed off to the team to own and run after deployment

Code and documentation are being organized here.

## Stack

Web platform · data centralization · client-format export
